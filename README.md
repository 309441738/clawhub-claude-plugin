# 邮轮产品技能 (ClawHub) · Claude Code 插件

在 ClawHub 平台上提供邮轮产品信息，用户可以通过此技能获取最新邮轮产品详情。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/clawhub-claude-plugin
/plugin install clawhub@clawhub-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **邮轮产品技能 (ClawHub)**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
